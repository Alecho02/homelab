# Screenshots

Place exported public screenshots here when you want to commit them.

Recommended filenames:

- `homepage-overview.png`
- `grafana-nas-apps.png`
- `grafana-observability.png`
- `cluster-portal.png`

Keep the images free of:

- API keys
- private IPs
- usernames
- hostnames that should stay private
- terminal sessions with secrets

These screenshots are intended to show the shape of the homelab, not any sensitive operational detail.
