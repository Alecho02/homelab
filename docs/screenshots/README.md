# Screenshots

This folder stores public captures from the live homelab. The images should stay free of secrets, private IPs, usernames, and any operational detail that should not be public.

## Included Captures

- `jellyseerr-dashboard.jpg` - Jellyseerr home view
- `jellystat-analytics.jpg` - Jellystat statistics view

## Guidance

- Export screenshots from the live apps
- Keep the frames clean and readable
- Avoid terminal windows, secrets, or private endpoints
- Use these images to show the shape of the platform, not the sensitive plumbing behind it
